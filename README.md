# College Appointment System 🏫📅

A backend system to manage appointment bookings between students and professors.

### 📦 Tech Stack
- Node.js + Express
- MongoDB + Mongoose
- Jest + Supertest (E2E tests)

### 📌 Features
- Professors set available time slots
- Students view and book appointments
- Professors can cancel bookings
- End-to-end test included

### ✅ Run Locally

```bash
npm install
node app.js
